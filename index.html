Research (dblp)

Thesis: Logic Programs for Temporal Graphs

A temporal graph evolves in time by adding, removing and changing nodes and edges. It serves as input to a logic program, which uses logical inference to obtain implicit information and answer a query.

The "depth" of the logical inference determines how fast the answer to the query is computed. With Alessandro Artale, Michael Zakharyaschev and Vladislav Ryzhikov, we work on identifying the features of individual programs that make them difficult to compute and how to optimize those features.
